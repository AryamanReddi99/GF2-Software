    E is NOR

    F has 4 inputs
}

    CKL1 is a clock

}

    }

        S1 to B.1
        S3 to B.2
    }
    Device C: {
        S2 to C.1
        B to C.2
    }
    Device D: {
        A to D.1
        B to D.2
    }
    Device E: {

    }

        E to G.1

        C to H.1
